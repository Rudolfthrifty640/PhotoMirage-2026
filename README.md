## 🚀 Getting started

Once you install the program, you can find it in your Start menu. Click the Windows icon on your taskbar and type "PhotoMirage-2026." Press Enter to open the application.

### Basic workflow

1. Import your photo. Click "File" and then "Open." Select the image you want to animate.
2. Use the "Motion Arrows" tool. Drag these arrows over parts of the photo you want to move.
3. Define "Anchor Points." Place these markers on areas of the image you want to keep still, such as buildings or rocks.
4. Adjust the speed. Use the slider on the right side of the screen to make the movement faster or slower.
5. Preview your work. Click the "Play" button under the image window to see the animation in real-time.
6. Export your project. When you feel satisfied with the result, click "Export." Choose your desired file format, such as `.mp4` or `.gif`, and save your new animation.

## 💡 Tips for better results

Good results depend on the quality of your source image. Use high-resolution photos for the best output. 

Focus on natural movement when placing your motion arrows. For example, arrows should follow the direction of flowing water or blowing hair. Do not overwhelm your image with too many arrows. A few well-placed arrows look better than many messy ones.

Use anchor points to stabilize the edges of your photo. If you animate a cloud, anchor the sky around it to prevent the entire image from warping.

Save your project often. Use the "Save Project" option under the "File" menu to store your work in a format that you can change later. The software creates a project file that stores your motion arrows and anchor points separately from the exported video.

## 🛠 Troubleshooting common issues

If the application fails to open, check the following points:

* Ensure you installed the latest Windows updates. Outdated system files often block new software.
* Run the installer as an administrator. Right-click the installation file and select "Run as administrator."
* Close heavy applications like video editors or browsers if the program runs slowly. These apps consume significant RAM and processing power.
* Reinstall the software. If you experience crashes, delete the current install and run the installer again.

If you encounter issues during the export process, ensure you have enough disk space on your computer. Large animation files require significant free storage. Check your file path to make sure you have permission to save files in that specific folder. If you choose a protected system folder, the program might fail to save your file. Select a user-accessible folder like "Videos" or "Documents" instead.
